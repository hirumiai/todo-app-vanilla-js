# TugasKu - Aplikasi To-Do Sederhana

Aplikasi web To-Do List sederhana yang dibuat untuk melatih fundamental JavaScript (Vanilla JS).

## Fitur
- Menambah tugas baru
- Menandai tugas sebagai selesai (dengan klik)

## Teknologi yang Digunakan
- HTML5
- CSS3
- JavaScript (ES6+)
